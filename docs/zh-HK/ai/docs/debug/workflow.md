# 工作流調試（整体調試）

## 功能说明

Agent Builder 支援对 **整个工作流进行調試运行**，
在不發布 Agent 的前提下，完整執行从開始節點到终止節點的流程。

工作流調試用于验证：

* 流程结构是否正确
* 節點之间的连接与執行顺序
* 数据在流程中的传递是否符合预期


## 核心特点

* **無需發布 Agent**
* 使用当前畫布中的最新版本
* 按真实執行路径完整运行流程
* 可观察每个節點的執行结果


## 适合的使用场景

* 验证流程整体邏輯是否闭环
* 检查条件分支是否按预期触发
* 調試多節點組合、複雜控制流程
* 發布前的完整流程验证


## 使用建議

* 在流程搭建完成后进行整体調試
* 修改任一節點后，建议重新調試流程
* 结合单節點調試，逐步排查複雜问题


## 一句话总结

> **工作流調試用于“验证整体”，
> 是發布前必不可少的一步。**

## 两种調試方式的关系（理解即可）

* **单節點調試**：
  解决"这个節點对不对"

* **工作流調試**：
  解决"这些節點连在一起对不对"
