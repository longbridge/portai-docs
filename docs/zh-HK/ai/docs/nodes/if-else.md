# If-Else

## 是什麼
条件判斷節點用于**根据已有变量进行判斷，并决定流程接下来走哪一条路径**。
它让流程具备“选择”和“分支”能力，而不是只能线性執行。

## 您可以用它做什麼

#### 1. 根据结果走不同流程

你可以基于上游節點的輸出：

* 判斷文本内容
* 判斷变量是否存在
* 判斷数值大小或状态

根据判斷结果，让流程进入不同的分支。

例如：

* 判斷用户问题类型
* 判斷模型是否返回有效结果
* 判斷是否需要调用工具

---

#### 2. 構建多分支流程

条件判斷節點可以：

* 配置多个判斷条件
* 每个条件对应一条分支路径
* 未命中条件时走默认分支

这让你可以構建更複雜、更灵活的流程。

## 条件配置說明

* 条件通常基于流程变量进行判斷
* 支援常见判斷方式，如：

  * 等于 / 不等于
  * 包含 / 不包含
  * 是否为空
  * 数值比较（大于、小于等）

每个条件都会连接到一个下游節點。

## 使用規則

* 条件判斷節點必须有至少一个下游分支
* 每条分支都只能在条件满足时才会被執行
* 条件本身不产生新数据，只负责“判斷和分流”

## 使用建議

* 条件尽量清晰，避免多个条件重叠
* 为複雜判斷设置一个兜底分支，防止流程中断
* 在进入条件判斷前，确保相关变量已准备好

## 典型場景

* 根据用户意图选择不同处理流程
* 判斷是否需要再次调用模型
* 判斷工具调用是否成功
* 多轮流程中的决策点
