# Code 節點

## 是什麼

Code 節點用于**編寫并執行自定义代碼**，对流程中的数据进行处理、转换或计算。

它适合处理**规则明確、邏輯确定、模型不擅长或不应参与的任務**。

## 您可以用它做什麼

#### 1. 处理和转换数据

使用 Code 節點可以：

* 清洗和格式化数据
* 转换数据结构（如 JSON 转换、字段映射）
* 合并、拆分或重组变量

例如：

* 将模型輸出整理成标准格式
* 对工具返回结果做二次处理

---

#### 2. 執行确定性邏輯

Code 節點非常适合：

* 条件判斷
* 数值计算
* 规则校验
* 固定算法处理

这些任務比交给大模型更稳定、更可控。

---

#### 3. 作為流程“粘合层”

在複雜流程中，Code 節點常用于：

* 连接多个節點的輸出
* 统一不同来源的数据格式
* 为后续節點准备干净、可用的输入

## 输入与輸出

* **输入**：

  * 来自上游節點的变量
* **輸出**：

  * 代碼執行后的结果
  * 可作為后续節點的输入或判斷条件

## 使用規則

* Code 節點只執行你編寫的代碼邏輯
* 不具备模型推理能力
* 執行失败会导致流程中断（需配合条件或兜底邏輯）

## 使用建議

* 只放确定性邏輯，不要在 Code 節點中模拟“思考”
* 代碼尽量简洁，避免複雜依赖
* 对关键輸出做好异常处理和默认值

## 典型場景

* 数据清洗与转换
* 规则判斷与校验
* 结构化輸出生成
* 模型与工具结果的中间处理
