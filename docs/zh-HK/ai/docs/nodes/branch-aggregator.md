# 分支聚合器

## 是什麼

分支聚合器用于**将来自不同分支的结果重新汇合到一起**，让流程在分支之后可以继续统一執行。

你可以把它理解为：

> **“流程的合流点。”**

## 您可以用它做什麼

#### 1. 合并不同分支的執行结果

在条件判斷或分类之后：

* 不同分支可以产生不同结果
* 分支聚合器负责接收这些结果
* 将流程重新拉回到一条主线

---

#### 2. 统一后续处理邏輯

使用分支聚合器后：

* 不需要在每个分支重复配置相同節點
* 后续節點只需连接一次

这能显著减少流程重复。

## 输入与輸出

* **输入**：

  * 来自不同分支的变量或结果
* **輸出**：

  * 聚合后的统一结果
  * 可被后续節點使用

## 使用規則

* 所有需要汇合的分支都应连接到同一个聚合器
* 聚合器本身不做判斷
* 聚合规则需明確（如选择某个结果、合并字段等）

## 使用建議

* 分支輸出结构尽量保持一致
* 聚合前避免分支变量命名冲突
* 聚合器后统一做模板转换或輸出

## 典型場景

* 条件分支后的统一輸出
* 多路径处理后的结果汇总
* 分类流程结束后的统一回覆
