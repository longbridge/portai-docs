# 迭代節點（Iteration）

## 是什麼

迭代節點用于**对一组数据进行逐项处理**。
它会把一份“集合型数据”（如数组、列表）拆分成多个元素，**按顺序重复執行同一段子流程**。

你可以把它理解为：

> **“对列表里的每一项，都走一遍同样的流程”**

## 您可以用它做什麼

#### 1. 批量处理数据

迭代節點非常适合：

* 遍历列表
* 批量生成结果
* 对每一项執行相同邏輯

例如：

* 对多个问题分别生成答案
* 对搜索结果逐条分析
* 对一组数据做格式转换

---

#### 2. 将单个流程复用到多项数据

你只需要设计一次处理邏輯，
迭代節點会帮你自动重复執行。

## 工作方式（理解即可）

1. 输入一个列表类型的变量
2. 每次取出其中一个元素
3. 執行连接在迭代節點内的子流程
4. 收集每一次執行的结果
5. 輸出为新的结果集合

## 输入与輸出

* **输入**：

  * 列表 / 数组类型的数据
* **輸出**：

  * 每一轮執行产生的结果集合

## 使用規則

* 迭代節點必须接收集合类型输入
* 子流程中不要修改原始集合结构
* 每一项迭代邏輯应保持一致

## 使用建議

* 单次迭代邏輯尽量簡單
* 避免在迭代中嵌套複雜 Agent
* 对大规模列表注意執行成本

## 典型場景

* 批量内容生成
* 批量分析与处理
* 列表型数据的逐项執行
