# 知識庫功能簡介

在 Infra 平台中，你可將自有資料作為「知識」整合到 AI 應用中。知識為大語言模型（LLM）提供領域上下文，使回覆更準確、相關，並減少幻覺。該能力基於檢索增強生成（RAG）技術實現。

## RAG 工作原理

LLM 不限於預訓練資料，還會將你的自訂知識作為額外事實來源：

| 環節 | 說明 |
|------|------|
| **檢索** | 系統從已整合的知識庫中檢索與問題最相關的資訊。 |
| **增強** | 檢索結果與用戶問題一併作為增強上下文發送給 LLM。 |
| **生成** | LLM 基於該上下文生成更精準的答案。 |

知識以檔案形式上傳至知識庫並儲存。你可建立多個知識庫，按領域、場景或資料來源劃分，並按需接入應用。

## 應用場景

知識庫適用於基於自有資料與領域知識的 AI 應用。典型場景包括：

- **智能客服**：基於產品文件、FAQ、故障排查指南等，智能回答客戶問題。
- **投資風格分析**：基於特定投資大師的持倉與歷史資料，生成風格分析報告，供大模型做風格化投資決策。
- **財報分析**：上傳歷年財報、同業財務資料等，構建金融資料知識庫，支撐大模型進行財報與對比分析。

## 建立知識庫

- **快速建立**：上傳檔案、設定處理規則後，按嚮導點擊「繼續」即可完成建立。
- **建立空白知識庫**：可先建立僅含名稱的空白知識庫，後續再上傳並匯入資料。

## 管理與優化知識庫

- **維護文件與分段**：對文件及分段進行檢視、新增、修改、刪除，保持內容最新、準確、一致。
- **召回測試**：透過模擬提問，驗證知識庫的召回效果。
- **調整設定**：按需調整索引方式、嵌入模型、檢索策略等參數。

## 使用知識庫

透過應用中的「知識檢索」節點，將指定知識庫接入你的 AI 應用，為對話或任務提供上下文。
