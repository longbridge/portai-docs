# Web App（Chatbot 應用）

## 功能说明

Web App 是平台提供的 **统一 Chatbot 使用入口**，
用于以對話形式使用已發布的 ChatFlow Agent。

用户無需进入 Agent Builder，即可直接与 Agent 进行交互。


## 访问方式

在 **PortAI 首頁**，你可以：

* 进入 Chatbot Web App
* 以聊天窗口的形式使用不同的 ChatFlow Agent


## 應用广场中的 Chatbot

在 Web App 中，你可以：

* 打开 **應用广场**
* 浏览平台中已公开的 Chatbot
* 选择任意 Chatbot 进行聊天和使用


## 常用 Chatbot

### 自动加入常用

* 使用过的應用广场 Chatbot
* 会自动出现在 **常用 Chatbot** 列表中
* 方便下次快速访问

## 手动固定

* 你也可以将任意 Chatbot
* **手动固定到常用 Chatbot**
* 用于长期或高频使用


## 使用特点

* Web App 仅支援使用，不支援编辑
* 使用体验以對話为主，贴近真实 Chatbot 场景
* 不同 Chatbot 之间互不影响


## 使用建議

* 将高频使用的 Chatbot 固定到常用列表
* 透過應用广场探索更多可用 Agent
* 对外使用场景优先透過 Web App 提供能力


## 一句话总结

> **Web App 是 Agent 的“使用层”，
> 让 ChatFlow Agent 以最自然的方式被访问和使用。**
