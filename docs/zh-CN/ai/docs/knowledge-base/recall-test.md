# 召回测试

通过召回测试，可模拟用户提问，验证知识库的召回效果，并尝试不同检索设置以找到最佳配置。

## 进入召回测试

在**知识库文档列表页**或**知识库设置页**，点击右上角「召回测试」图标，进入测试页面。

## 使用说明

| 项目       | 说明 |
|------------|------|
| 测试方式   | 输入问题，查看系统从知识库中召回的文本分段及排序结果。 |
| 设置作用范围 | 在召回测试中修改的检索设置，仅在当前测试会话中临时生效，用于对比不同检索策略与参数的效果。检索参数含义见「创建知识库」中的检索策略说明。 |
| 历史记录   | 本页会记录该知识库相关的召回测试记录，包括检索方式、查询文本及时间。支持点击历史问句快速复制，便于重复测试或对比。 |

## 建议用法

- 用业务中的典型问题做测试，观察召回片段是否覆盖所需信息。
- 在测试中切换不同检索策略（向量、全文、混合）及 TopK、Score 阈值等参数，对比召回质量与数量。
- 结合历史记录，对同一问题在不同设置下的结果进行对比，确定适合该知识库的配置。
