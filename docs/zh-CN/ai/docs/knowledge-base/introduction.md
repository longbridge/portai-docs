# 知识库功能简介

在 Infra 平台中，你可将自有数据作为「知识」集成到 AI 应用中。知识为大语言模型（LLM）提供领域上下文，使回复更准确、相关，并减少幻觉。该能力基于检索增强生成（RAG）技术实现。

## RAG 工作原理

LLM 不限于预训练数据，还会将你的自定义知识作为额外事实来源：

| 环节 | 说明 |
|------|------|
| **检索** | 系统从已集成的知识库中检索与问题最相关的信息。 |
| **增强** | 检索结果与用户问题一并作为增强上下文发送给 LLM。 |
| **生成** | LLM 基于该上下文生成更精准的答案。 |

知识以文件形式上传至知识库并存储。你可创建多个知识库，按领域、场景或数据源划分，并按需接入应用。

## 应用场景

知识库适用于基于自有数据与领域知识的 AI 应用。典型场景包括：

- **智能客服**：基于产品文档、FAQ、故障排查指南等，智能回答客户问题。
- **投资风格分析**：基于特定投资大师的持仓与历史数据，生成风格分析报告，供大模型做风格化投资决策。
- **财报分析**：上传历年财报、同业财务数据等，构建金融数据知识库，支撑大模型进行财报与对比分析。

## 创建知识库

- **快速创建**：上传文件、设置处理规则后，按向导点击「继续」即可完成创建。
- **创建空白知识库**：可先创建仅含名称的空知识库，后续再上传并导入数据。

## 管理与优化知识库

- **维护文档与分段**：对文档及分段进行查看、添加、修改、删除，保持内容最新、准确、一致。
- **召回测试**：通过模拟提问，验证知识库的召回效果。
- **调整设置**：按需调整索引方式、嵌入模型、检索策略等参数。

## 使用知识库

通过应用中的「知识检索」节点，将指定知识库接入你的 AI 应用，为对话或任务提供上下文。
