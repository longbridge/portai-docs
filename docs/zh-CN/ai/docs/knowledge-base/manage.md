# 管理知识库

本文说明如何在知识库中管理文档、分段及知识库设置。

## 管理文档

每个导入的本地文件在知识库中以「文档」形式存在。你可在文档列表中查看和管理所有文档，按需保留或移除，并根据上传时间判断时效性。

文档列表支持以下操作：

| 操作     | 说明 |
|----------|------|
| 添加文件 | 在文档列表中点击「添加文件」进入上传流程，可选择文件、设置分段并查看处理进度。分段设置目前仅能在上传或重新上传时修改，文档管理中暂不支持单文档分段设置编辑。 |
| 删除文档 | 永久删除该文档。不可撤销、不可恢复。 |
| 启用/禁用 | 临时将该文档纳入或排除在检索范围之外。 |
| 编辑     | 进入文档详情，对分段内容进行编辑。详见下文「管理分段」。 |
| 重命名   | 修改该文档的显示名称。 |

## 管理分段

文档按分段设置被拆分为一个或多个「分段」，分段是检索的基本单位。你可在文档的分段列表中查看和管理分段，以提升检索效率与准确性。

分段列表支持以下操作：

| 操作           | 说明 |
|----------------|------|
| 添加分段       | 在文档详情页新增分段，可同时设置分段关键词，便于后续检索匹配。 |
| 删除分段       | 永久删除该分段。不可撤销、不可恢复。 |
| 启用/禁用      | 临时将该分段纳入或排除检索。可通过分段前的圆圈批量勾选后操作。已禁用的分段仍可编辑。 |
| 添加/编辑/删除关键词 | 为分段维护关键词，提升该分段的可检索性。 |

## 分段设置实践建议

分段完成后，建议逐段检查：语义完整、未被强制截断、长度适中，以保证检索准确性与相关性。

常见问题与影响：

| 情况         | 影响 |
|--------------|------|
| 分段过短     | 上下文不足，易造成语义丢失和回答不准确。 |
| 分段过长     | 易包含无关内容，引入噪音、降低检索精度。 |
| 语义不完整   | 句子或段落被强行切断，易导致检索结果缺失或误导。 |

## 知识库设置

在知识库页点击右上角「知识库设置」进入设置页。

支持的操作包括：

| 操作           | 说明 |
|----------------|------|
| 修改名称与描述 | 自定义知识库的名称与描述，用于标识与区分。名称不可与其他知识库重复，否则修改将失败。 |
| 修改检索设置与模型 | 调整知识库的检索方式及所用模型、参数。具体选项与含义参见「创建知识库」中关于检索策略的说明。 |
