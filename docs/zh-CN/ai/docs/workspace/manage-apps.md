# 应用管理（My Agent）

## 功能说明

**My Agent 页面**用于管理你在当前工作空间下的所有 Agent 应用。

这是你创建、查看和进入 Agent 的主要入口。


## 应用列表

在 My Agent 页面中，你可以看到：

* 当前工作空间下的 **Agent 应用列表**
* 每个应用代表一个独立的 Agent
* 不同工作空间的应用彼此隔离、互不影响


## 支持的操作

在 My Agent 页面中，你可以：

* **创建 Agent**

  * 新建一个 Agent 应用
* **删除 Agent**

  * 删除当前工作空间下的 Agent
* **快速进入 Agent**

  * 直接进入 Agent Builder，进行编辑、调试或查看


## 应用与工作空间的关系

* 每个 Agent **只能隶属于一个工作空间**
* Agent 无法跨工作空间共享或移动
* 切换工作空间后，看到的是该空间下的应用列表


## 使用提示

* 在创建 Agent 前，请确认当前所处的工作空间
* 删除 Agent 操作不可逆，请谨慎操作
* 不同工作空间可用于区分不同项目或用途
