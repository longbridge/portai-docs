
# 分享功能

## 功能说明

分享功能用于将 **已发布的 ChatFlow Agent** 提供给更多用户使用。

通过分享，你可以让 Agent：

* 仅供指定范围使用
* 或对所有人开放使用


## 支持的分享方式

### 公开使用

当一个 ChatFlow Agent 被设置为 **公开** 后：

* **任何人都可以使用该 Agent**
* 不再受工作空间成员限制
* 适合通用型、示例型或工具型 Agent

### 发布到应用广场

在公开的基础上，你可以选择将 ChatFlow Agent：

* **发布到应用广场**
* 供其他用户浏览、选择和使用

应用广场中的 Agent：

* 可被不同用户直接使用
* 不影响原有工作空间和编辑权限


## 使用说明

* 仅 **已发布的 ChatFlow Agent** 支持分享
* 分享不会暴露 Agent 的内部流程和配置
* 使用者仅能进行使用，不具备编辑权限


## 使用建议

* 在公开前，确认 Agent 行为稳定、输出合理
* 通用能力型 Agent 更适合发布到应用广场
* 内部或实验性 Agent 建议仅限私有使用


## 一句话总结

> **分享功能让 ChatFlow Agent 从“个人工具”，
> 变成“可被复用的公共能力”。**
