# 工作流调试（整体调试）

## 功能说明

Agent Builder 支持对 **整个工作流进行调试运行**，
在不发布 Agent 的前提下，完整执行从开始节点到终止节点的流程。

工作流调试用于验证：

* 流程结构是否正确
* 节点之间的连接与执行顺序
* 数据在流程中的传递是否符合预期


## 核心特点

* **无需发布 Agent**
* 使用当前画布中的最新版本
* 按真实执行路径完整运行流程
* 可观察每个节点的执行结果


## 适合的使用场景

* 验证流程整体逻辑是否闭环
* 检查条件分支是否按预期触发
* 调试多节点组合、复杂控制流程
* 发布前的完整流程验证


## 使用建议

* 在流程搭建完成后进行整体调试
* 修改任一节点后，建议重新调试流程
* 结合单节点调试，逐步排查复杂问题


## 一句话总结

> **工作流调试用于“验证整体”，
> 是发布前必不可少的一步。**

## 两种调试方式的关系（理解即可）

* **单节点调试**：
  解决"这个节点对不对"

* **工作流调试**：
  解决"这些节点连在一起对不对"
