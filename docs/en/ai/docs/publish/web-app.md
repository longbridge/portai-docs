# Web App (Chatbot Application)

## Function Description

The Web App is a **unified Chatbot entry point** provided by the platform, used to interact with published ChatFlow Agents in a dialogue format.

Users can interact with Agents directly without entering the Agent Builder.


## Access Method

On the **PortAI homepage**, you can:

* Enter the Chatbot Web App
* Use different ChatFlow Agents in the form of a chat window


## Chatbots in the App Marketplace

In the Web App, you can:

* Open the **App Marketplace**
* Browse the public Chatbots on the platform
* Select any Chatbot to chat with and use


## Common Chatbots

### Automatically Added to Common

* Chatbots from the App Marketplace that you have used
* Will automatically appear in the **Common Chatbots** list
* Convenient for quick access next time

## Manual Pinning

* You can also **manually pin** any Chatbot
* To the Common Chatbots list
* For long-term or high-frequency use


## Usage Characteristics

* The Web App only supports usage, not editing
* The experience is centered around dialogue, close to a real Chatbot scenario
* Different Chatbots are independent of each other


## Usage Recommendations

* Pin high-frequency Chatbots to the common list
* Explore more available Agents through the App Marketplace
* Prioritize providing capabilities via the Web App for external usage scenarios


## One-sentence Summary

> **The Web App is the "usage layer" of an Agent, allowing ChatFlow Agents to be accessed and used in the most natural way.**
